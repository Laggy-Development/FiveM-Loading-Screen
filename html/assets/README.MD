Zet al je afbeeldingen in DEZE map (html/assets/).
Hieronder staat per bestand wat je nodig hebt:

──────────────────────────────────────────────────────
  VERPLICHT
──────────────────────────────────────────────────────

background.jpg
  → De achtergrondafbeelding van het volledige scherm.
  → Aanbevolen: 1920×1080 px, JPG of WebP.
  → Tip: een donkere/sfeervolle screenshot van jouw server werkt perfect.

──────────────────────────────────────────────────────
  KAARTEN (linkerkant)
──────────────────────────────────────────────────────

card-store.png
  → Karakter- of item afbeelding rechts in de Store kaart.
  → Aanbevolen: PNG met transparante achtergrond, ~300×400 px.
  → Laat de onderkant van het karakter zichtbaar aansluiten op de kaart.

card-changelog.png
  → Karakter of item afbeelding rechts in de Changelog-kaart.
  → Zelfde richtlijnen als card-store.png.

──────────────────────────────────────────────────────
  RECHTER PANEEL
──────────────────────────────────────────────────────

dc-avatar1.png  /  dc-avatar2.png  /  dc-avatar3.png
  → Kleine Discord-profielfoto's (32×32 px of groter, wordt verkleind).
  → Gebruik avatars van Discord-medewerkers of actieve leden.

discord-logo.png
  → Het Discord-logo (wit, transparante achtergrond).
  → Standaard Discord Blurple logo, 18×18 px weergave.

last-player.png
  → Profielfoto van de laatst online speler (of een standaard avatar).
  → 46×46 px weergave, vierkant bijgesneden.

potm-avatar.png
  → Foto/avatar van de Player of the Month.
  → 80×80 px weergave; portretfoto werkt het beste.
